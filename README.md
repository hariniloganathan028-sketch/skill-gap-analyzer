# Skill Gap Analyser

## Overview
Skill Gap Analyser is a web application that compares a student's existing skills with the required skills for a selected job role and identifies the missing skills.

## Features
- Student Login
- Skill Analysis
- Gap Percentage
- Recommended Skills
- Dashboard
- Report Generation

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- SQLite
- GitHub

## Deployment Stack
Frontend : HTML, CSS, JavaScript
Backend : Node.js + Express.js
Database : SQLite
Version Control : Git & GitHub
